# Задание «КТ №1 - Бэкэнд для todo-листа»

Написать бэкэнд для списка задач.

Требуется написать следующий контроллер:
1) Метод, возвращающий список задач с короткой информацией
2) Метод, возвращающий полную информацию о конкретной задаче
3) Метод создания задачи
4) Метод изменения задачи
5*) Метод, делающий задачу выполненой
6) Метод удаления задачи из списка

Пример задачи:

*Важные данные*
Guid id,
string Name,
bool isComplited,
DateTime Deadline,

*Дополнительные данные*
DateTime CreatedAt,
DateTime UpdatedAt,
string Description
